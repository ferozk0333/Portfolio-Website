Mental Health Analytics with NLP

GitHub

All code is available on my GitHub.

Problem Statement

Mental health data is often unstructured and difficult to analyze. Understanding sentiment, emotions, and mental health trends from text data is crucial to addressing the growing mental health crisis.

Proposed Solution

This project leverages NLP techniques such as sentiment analysis, VADER, and RNNs to extract insights from mental health-related texts. It provides visualizations and actionable insights using Power BI.

Business Domain

Healthcare, Mental Health, and Data Analytics.

Tech Stack

  • Flask
  • SQL
  • React
  • Azure
