Mid Day Meal Analytics using Machine Learning

Problem Statement

The Mid-Day Meal Scheme, serving over 120 million school children in India, faces challenges like inconsistent meal quality, hygiene issues, and fraudulent attendance reporting. These inefficiencies hinder the programâ€™s goal of improving nutrition, education, and social equity, and the lack of scalable monitoring solutions exacerbates the problem.

Proposed Solution

Leveraging machine learning and data analytics, the project integrates supervised learning (e.g., Random Forest) for meal quality and hygiene prediction and unsupervised learning (e.g., DBSCAN) for anomaly detection in attendance and meal distribution. A streamlined pipeline, supported by tools like Pandas, SQL, Streamlit, and Power BI, provides real-time insights through interactive dashboards, enabling transparency, efficiency, and targeted interventions.
